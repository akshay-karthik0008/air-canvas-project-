# air-canvas-project-

Air Canvas is an interactive virtual drawing system developed using Python and OpenCV. It enables users to write or sketch in the air without touching a physical surface. The application captures live video from a webcam, detects the colored object or pen tip, and tracks its movement to create drawings on a virtual screen.

The project uses computer vision techniques such as color detection, contour tracking, and image processing to provide smooth and real-time interaction. It supports multiple drawing colors, erasing functionality, and touch-free operation, making it suitable for educational, creative, and smart interaction applications.

Features
Real-time air drawing
Pen tip / color tracking
Multiple brush colors
Eraser functionality
Smooth virtual canvas
Webcam-based interaction
Lightweight and easy to use
